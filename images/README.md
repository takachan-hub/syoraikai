## ヒーロー背景画像について

ヒーロー背景画像は CSS で設定しています。
変更する場合は css/style.css の以下の箇所を編集してください：

```css
.hero-bg {
  background-image: url('../images/hero-bg.jpg');
  ...
}
```

## 注意事項

- ファイル形式は .jpg / .jpeg / .png / .webp のいずれも使用できます
- ファイル名を変更した場合は index.html 内の src 属性も合わせて修正してください
- 画像は正方形または指定の縦横比に近いほどレイアウトが整います
